## Overview
This project analyzes customer lead generation and conversion performance
based on real internship data collected at MyWish Marketplaces Pvt. Ltd. (Wishfin).

## Objectives
- Analyze weekly lead performance
- Calculate conversion rates
- Identify trends & improvement areas
- Visualize business insights

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
Weekly data including:
- Leads Generated
- Leads Contacted
- Qualified Leads
- Converted Leads

## Key Insights
- Avg leads per week: ~700
- Avg conversion rate: ~27%
- Best week: Week 6
- Timely follow-ups increase conversion
